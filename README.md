# RefactoringSheet

Intellij Plugin , 
重构方法速查。《重构-改善既有代码的设计》（Refactoring, Improving the Design of Existing Code）

菜单位于 "Tools" 下。
鼠标悬浮时自动展开重构方法列表。
鼠标点击跳转到电子书对应的章节。
